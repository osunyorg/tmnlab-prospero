Dans le contexte SEO, le ranking ou rang est la position qu’occupe une page ou un site web dans les pages de résultats d’un outil de recherche qui sont consécutives à une requête effectuée sur un mot clé. On utilise également le terme de positionnement SEO.

On parle surtout de ranking pour le référencement naturel, mais le terme peut être également utilisé pour les liens commerciaux.
