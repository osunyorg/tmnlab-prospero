Enregistrement d'un site web dans les annuaires, les moteurs de recherche, les portails spécialisés pour en permettre l'apparition dans les résultats de recherche des utilisateurs.

Il existe deux types de référencement, le SEA (search engine advertisement), dit payant ou sponsorisé, et le SEO (search engine optimization) dit naturel ou organique, qui  repose sur trois grands piliers : la technique, le contenu et la popularité.
